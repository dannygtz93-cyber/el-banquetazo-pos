# 🌮 El Banquetazo POS

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Version](https://img.shields.io/badge/version-0.2.0-green)
![Status](https://img.shields.io/badge/status-development-orange)
![License](https://img.shields.io/badge/license-MIT-lightgrey)

Aplicación de punto de venta (POS) para taquería desarrollada en **Python + Flet + SQLite**.

Diseñada específicamente para un puesto de tacos con operación rápida y sencilla.

---

# 🚀 Características

* Registro rápido de órdenes
* Múltiples órdenes abiertas
* Cierre de orden al cobrar
* Pagos en:

  * 💵 Efectivo
  * 💳 Tarjeta
  * 📱 Transferencia
* Interfaz táctil optimizada para tablet
* Configuración de menú y precios
* Base de datos local (SQLite)
* Sin necesidad de internet

---

# 🧠 Objetivo

Crear una aplicación ligera, rápida y fácil de usar para taquerías pequeñas y medianas.

---

# 🏗️ Tecnologías

* Python 3.11+
* Flet
* SQLite
